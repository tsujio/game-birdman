Birdman Challenge (2021)

鳥人間コンテストを題材にしたタップ操作だけで遊べるゲームです。

ワンボタンゲームをたくさん作ったので、その作り方をおさらいしたい - ABAの日誌 の記事でワンボタンゲームというジャンルがあることを知り、クリック／タップ操作だけが許された環境でどれだけ「おもしろさ」を追及できるかということに興味を持ったのがきっかけです。

ゲームを作って公開することは以前から興味がありましたが、おもしろいゲームを作るにはシナリオが壮大だったりイラストや音楽を凝ったりしないといけないのではないかという思いからなかなか手が出ませんでした。しかしこのワンボタンゲームなら規模が小さくとも何か「おもしろさの本質」のようなものを発見し表現できるような気がして、ゲーム作りに挑戦しようと思えるようになりました。

実装は Ebiten というゲームライブラリを用いました。Web や iOS/Android 向けのバイナリを同じソースから生成できるというのが利点です。
